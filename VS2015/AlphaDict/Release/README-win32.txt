
1) Run AlphaDict.exe.

2) Put dictionaries at "./system/dicts".

3) AlphaDict will create a user directory at "C:\Users\xxx\AppData\Roaming\AlphaDict" saving
   user files like configure, log, etc.
   xxx is the account name, this path may be a little different between windows platforms.
   If the program initialization failure, check alphadict.log, delete configure.xml and try again.

4) You can download more dictionaries from "http://pan.baidu.com/s/1dD9v12D"


Read "doc/README.en" get more information.
